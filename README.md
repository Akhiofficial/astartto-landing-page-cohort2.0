# Astratto Landing Page

A modern, visually stunning landing page for a creative agency, built with React and Tailwind CSS. This project features high-quality 3D illustrations, interactive components, and a unique, vibrant design language.

## üöÄ Technologies Used

*   **React**: Core frontend library for building the user interface.
*   **Vite**: Next-generation frontend tooling for fast build and development.
*   **Tailwind CSS**: Utility-first CSS framework for rapid and custom styling.
*   **Lucide React**: Beautiful, consistent icons.
*   **ESLint**: Pluggable linting utility for JavaScript and JSX.

## ‚ú® Key Features

*   **Immersive Hero Section**: Features floating 3D assets with custom bounce animations and unique border-radius shapes.
*   **Interactive Services Grid**:
    *   **3D/2D Animation Cards**: Showcases services with high-quality imagery.
    *   **Cutout Effect**: A custom-designed center card with a transparent "cutout" notch revealing the background.
    *   **Hover Animations**: Smooth transitions and scale effects on interaction.
*   **Rich Typography About Section**:
    *   **Custom Styling**: Integrated vibrant text highlights (Orange, Purple, Pink).
    *   **Vector Decorations**: Custom SVG squiggles and arrows positioned precisely to match the design.
*   **Masonry Projects Layout**: A staggered 2-column grid showcasing recent work with reusable `ProjectCard` components.
*   **Dynamic Footer**: A layout-rich footer with a large CTA, spinning badges, and decorative elements.

## üé® Design Notes

The design follows a "Modern Whimsical" aesthetic, utilizing a color palette of Purple, Orange, and Pink against a clean white/zinc background. Custom border radiuses (`rounded-[60px]`, `rounded-bl-[50%]`) are used extensively to create a soft, approachable feel.
